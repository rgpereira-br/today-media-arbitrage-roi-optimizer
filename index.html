Configuração de API Key

A integração com a IA utiliza a API oficial da Gemini. Por segurança, sua chave é salva **apenas localmente no seu próprio navegador** (localStorage) e não é enviada para nenhum servidor externo.

Dica: Se você não inserir nenhuma chave de API, a ferramenta entrará em modo de simulação inteligente e gerará as análises com base em dados de amostra offline locais para avaliação rápida.
